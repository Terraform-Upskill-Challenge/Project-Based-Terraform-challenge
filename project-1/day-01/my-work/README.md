# What `terraform init` does

terraform init Initializing the backend, initializing providers plugins
# What`terraform plan` shows

terraform plan shows all the things terraform would change in the infrastructure

# What `terraform apply` does

terraform apply validate or apply all the changes in the infrastructure

# What the state file contains
The state file contains provider information (version, availability zone...), resource details(tags, instances...), and output values.

# Why state files are important
The state files is important, because it's help to nkow what do you want to change in the infrastruce
